# 🚀 DesignSync 실행 가이드

## 현재 상황

루트 디렉토리에서 `npm run dev`를 실행하려면 먼저 설정이 필요합니다.

## 빠른 실행 (3단계)

### 1️⃣ 루트 패키지 설치
```bash
npm install
```

### 2️⃣ 프론트엔드 & 백엔드 의존성 설치
```bash
./install.sh
```

### 3️⃣ 개발 서버 실행
```bash
npm run dev
```

끝! 이제 두 서버가 동시에 실행됩니다.

---

## 상세 설명

### 왜 오류가 났나요?

이전에는 루트 디렉토리에 `package.json`이 없었기 때문에 오류가 발생했습니다. 
이제 루트 디렉토리에도 `package.json`을 추가하여 한 번에 실행할 수 있게 했습니다.

### 실행 방법 비교

**이전 (별도 터미널 2개 필요):**
- 터미널 1: `cd frontend && npm run dev`
- 터미널 2: `cd backend && npm run dev`

**지금 (한 터미널에서 한 번에):**
- 루트에서: `npm run dev`

훨씬 편리합니다! 🎉
