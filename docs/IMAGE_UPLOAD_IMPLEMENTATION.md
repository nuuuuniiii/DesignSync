# 이미지 업로드 구현 가이드

## 🔍 현재 문제점

- 이미지가 My Project 화면에 표시되지 않음
- Cloudinary에 이미지가 업로드되지 않음
- 데이터베이스에 이미지 URL이 저장되지 않음

## 📋 구현 필요 사항

### 1. 백엔드: 이미지 업로드 API
- `POST /api/projects/:projectId/designs` - 디자인 생성
- `POST /api/projects/:projectId/designs/:designId/images` - 이미지 업로드
- 또는 프로젝트 생성 시 이미지도 함께 처리

### 2. 프론트엔드: 이미지 업로드 연동
- 프로젝트 생성 시 이미지 파일 전송
- FormData 사용하여 파일 업로드
- 업로드 진행 상태 표시

### 3. 데이터베이스: 이미지 URL 저장
- `design_images` 테이블에 이미지 URL 저장
- 프로젝트 목록 조회 시 이미지 URL 포함

## 🚀 구현 단계

### 단계 1: 이미지 업로드 API 구현
- 백엔드에서 Cloudinary 업로드 처리
- 이미지 URL을 데이터베이스에 저장

### 단계 2: 프로젝트 생성 시 이미지 업로드
- 프론트엔드에서 이미지 파일을 FormData로 전송
- 백엔드에서 받아서 Cloudinary 업로드

### 단계 3: 프로젝트 목록에 이미지 표시
- 프로젝트 목록 조회 시 이미지 URL 포함
- MyPage에서 이미지 표시
