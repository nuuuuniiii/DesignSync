## 주요 화면 목록

PRD 7번 섹션을 기반으로 한 화면 구성:

1. **Explore 화면** (7.1)
   - Web/Apps 선택
   - 카테고리 필터
   - Unsolved/Solved 필터
   - 프로젝트 카드 목록
   - Add New Project 버튼

2. **My Page 화면** (7.2, 7.3)
   - My Project 섹션
   - My Feedback 섹션

3. **프로젝트 등록 화면** (7.4)
   - 프로젝트 이름, 설명
   - App/Web 선택
   - 카테고리 선택
   - 피드백 타입 선택
   - 디자인 이미지 업로드
   - 피드백 이름 선택/입력

4. **New Feedback > Rating 화면** (7.5)
   - 별점 슬라이더 (1-5점)

5. **New Feedback > Screen 화면** (7.6)
   - Design 선택
   - Screen 선택
   - 피드백 질문 및 입력

6. **Project Overview 화면** (7.7)
   - 평균 별점
   - Designs별 화면 및 피드백
