# 이미지 업로드 구현 가이드

## 🔍 현재 문제점

이미지가 My Project 화면에 표시되지 않는 이유:

1. **로컬 미리보기만 함**: RegistrationPage에서 이미지를 선택하면 base64로 변환해서 로컬에서만 미리보기
2. **Cloudinary 업로드 안 함**: 실제로 Cloudinary에 업로드하지 않음
3. **프로젝트 생성 시 이미지 전송 안 함**: 프로젝트 생성 API 호출 시 이미지 데이터를 전송하지 않음
4. **데이터베이스 저장 안 함**: 이미지 URL이 데이터베이스에 저장되지 않음

## 💡 해결 방법

### 단계 1: 이미지 업로드 API 구현 (백엔드)
- Cloudinary에 이미지 업로드
- 데이터베이스에 이미지 URL 저장

### 단계 2: 프로젝트 생성 시 이미지 처리
- 프로젝트 생성 후 디자인 생성
- 각 디자인별 이미지 업로드 및 저장

### 단계 3: 프로젝트 목록에 이미지 포함
- 프로젝트 목록 조회 시 첫 번째 이미지 URL 가져오기
- MyPage에서 썸네일로 표시
