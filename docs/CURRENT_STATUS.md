5. ✅ 연결 테스트 성공

## ❌ 아직 구현되지 않은 기능

### 1. 로그인/인증 기능
- **현재 상태**: 로그인 기능이 구현되지 않음
- **GNB에 Sign In 버튼만 있음** (실제 로그인 기능 없음)
- 사용자 인증 시스템이 없음

### 2. 데이터 저장 API
- **프로젝트 생성 API** (`POST /api/projects`) - 미구현
- **프로젝트 조회 API** (`GET /api/projects`) - 미구현
- **피드백 생성 API** (`POST /api/feedbacks`) - 미구현
- **이미지 업로드 API** - 미구현

### 3. 프론트엔드-백엔드 연동
- 프론트엔드는 **Mock 데이터**만 사용 중
- 실제 API 호출 코드가 없음

## 📊 현재 구조

```
프론트엔드 (Mock 데이터 사용)
    ↓ (연동 안 됨)
백엔드 (테스트 API만 존재)
    ↓ (연결됨 ✅)
Supabase (테이블 구조만 생성됨)
```
